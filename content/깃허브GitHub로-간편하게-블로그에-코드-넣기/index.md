깃허브 Gist를 이용해 블로그에 코드를 쉽고 예쁘게 올릴 수 있다.

티스토리에도 코드 작성이 가능하길래 써봤는데 하이라이트가 하나도 안됐다ㅠㅠ

그래서 찾았는데 많은 블로그에서 추천하는 Color Sprinter는 이상하게 사이트 접속도 안됐다.

더 찾아본 결과 복붙만으로 쓸 수 있는건 깃허브였다!
