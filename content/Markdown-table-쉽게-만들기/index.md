# 마크다운 문법으로 테이블 쉽게 만들기

Tables Generator 사이트를 통해 보기 쉽게 틀을 만들 수 있다.  
작은 테이블을 만들 때는 그냥 만들어도 상관이 없으나 내용이 많은 테이블은 사이트로 만드는 게 더 편하다.
